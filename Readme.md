## Four-in-a-Row

This repo hosts both the client and server for the Four-in-a-Row game.

### Quick Links

- Client docs: [client_side/README.md](client_side/README.md)
- Server docs: [server_side/README.md](server_side/README.md)

### Repo Layout

- client_side: Vite/React front-end with Konva canvas board, Framer Motion FX, and Zustand state.
- server_side: Go module scaffold (`4_rows_backend`); endpoints still to be implemented.

### Running

- Client: see [client_side/README.md](client_side/README.md) for stack, structure, and dev commands.
- Server: see [server_side/README.md](server_side/README.md) for current status and next steps.
